Kinematics

Position

Position is a measure of where an object is located in space. It is often described using a set of coordinates, such as x, y, and z. The position of an object can change over time as it moves.

Time

Time is a measure of the duration between two events. In kinematics, time is often used to describe the rate at which an object moves. The change in position of an object over time is called its displacement.

Velocity

Velocity is a measure of how quickly an object is moving in a specific direction. It is often described using a vector that includes both speed and direction. The average velocity of an object can be calculated by dividing its displacement by the time it takes to travel that distance.

Acceleration

Acceleration is a measure of how quickly an object's velocity is changing. It is often described using a vector that includes both magnitude and direction. The average acceleration of an object can be calculated by dividing the change in velocity by the time it takes to make that change.
